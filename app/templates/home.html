import hdi_model
import pandas as pd

# Load deterministic ML pipeline
model = hdi_model.load_pipeline('models/HDI.pkl')

input_data = pd.DataFrame([{
    'LifeExpectancy': 82.5,
    'ExpectedSchooling': 17.2,
    'MeanSchooling': 13.0,
    'GrossNationalIncome': 65000
}])

prediction = model.predict(input_data)
print(f"Predicted HDI: {prediction[0]:.3f}")

Why does Linear Regression score ~1.00?
Because the HDI target label is synthesized using a deterministic rule-based heuristic (it is a geometric mean of normalized component indices), linear models are able to reconstruct the mathematical mapping almost perfectly on our cleaned dataset. Standard scaling does not modify the structural relationships, yielding identical performance.
